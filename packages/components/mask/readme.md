> 蒙层组件

- 所有弹窗类组件的基础组件

| 参数          | 说明             | 类型     | 默认值 |
| ------------- | ---------------- | -------- | ------ |
| open          | 是否显示 dialog  | boolean  | false  |
| show          | 打开弹窗         | function | -      |
| hiden         | 隐藏弹窗         | function | -      |
| mask-closable | 点击蒙层关闭弹窗 | boolean  | true   |
